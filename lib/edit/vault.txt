# File: vault.txt
#
# This file is used to create the "lib/data/vault.raw" file, which is
# used to initialize the "vault template" information for Sil.
#
# Do not modify this file unless you know exactly what you are doing,
# unless you wish to risk possible system crashes and broken savefiles.
#
#
# === Understanding vault.txt ===
#
# N: serial number : vault name
# X: room type : depth : rarity : rows : columns
# D: lines giving full layout of vault using symbols below
#
# 'N' indicates the beginning of an entry. The serial number must
#     increase for each new vault.
#
# 'X' is for extra information - room type, minimum depth, rarity,
#     rows, and columns.
#
#     type 10 - Gates of Angband
#     type 6  - interesting room
#     type 7  - lesser vault
#     type 8  - greater vault
#     type 9  - Morgoth's vault
#
# 'D' lines describe the layout of the vault. Lines must be padded
#     with spaces to fill the dimensions specified. Vaults are described
#     with the following symbols:
#
#     $ - outside of the vault, where corridors may be connected
#     # - granite
#     % - quartz vein
#     X - impenetrable rock
#     : - rubble
#     ; - glyph of warding
#     > - down stairs
#     < - up stairs
#     + - visible door
#     s - secret door
#     w - web
#     ^ - trap
#     0 - forge
#     1 - monster +1 level
#     2 - monster +2 levels
#     3 - monster +3 levels
#     4 - monster +4 levels
#     * - treasure +1 to +4 levels
#     & - treasure +1 to +4 levels (good)
#     ~ - a chest with +4 levels
#     S - an elf skeleton
#     ? - a monster +1 level and/or an item +1 level
#     @ - easterling spy
#     H - silent watcher
#     o - orc champion
#     O - orc captain
#     f - cat warrior
#     F - cat assassin
#     t - cave troll
#     T - troll guard
#     W - barrow wight
#     c - wolf +1d4 levels
#     v - vampire +1d4 levels
#     r - raukar +1d4 levels
#     M - spider +1d4 levels
#     d - dragon +4 levels
#     R - Gothmog
#     U - Ungoliant
#     D - Glaurung
#     G - Gorthaur
#     V - Morgoth
#     C - Carcharoth
#
#
# === Important notes ===
#
# * Note that the "spacing" in the "description" lines
#   is very important!
